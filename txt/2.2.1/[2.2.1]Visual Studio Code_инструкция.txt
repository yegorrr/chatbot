Инструкция по установке Visual Studio Code
(скрины взяты отсюда)

Чтобы загрузить и установить код Visual Studio, выполните следующие действия:

Откройте кодовую страницу Visual Studio. https://code.visualstudio.com/download
Нажмите кнопку «Windows», чтобы загрузить установщик.

Сохраните файл на свое устройство.
После скачивания запустите файл установки, следуя всем инструкциям установить среду на компьютер.
После открытия установленной студии, перейдите в раздел Расширения и найти расширение для Python.

Нажмите кнопку «установить» в расширении Pylance.

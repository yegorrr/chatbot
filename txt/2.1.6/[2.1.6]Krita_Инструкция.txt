Инструкция по установке фоторедактора «Krita».
Перейдите по ссылке: https://krita.org/en/
Нажмите на кнопку «Скачать».

По умолчанию появится установщик для Windows. Нажмите на эту кнопку.

Выберите папку, в которую хотите сохранить файл установки, и нажмите «Сохранить».
После скачивания запустите файл установки и следуйте всем инструкциям для установки Krita на компьютер.
Для начала выберите язык установки «English», интерфейс программы будет на русском языке.

Далее нажмите «Next».

Примите условия публичного соглашения, поставив галочку, и нажмите «Next».

Выберите папку для установки программы и нажмите «Next».

В следующем окне выберите функции, которые вы хотите установить. По умолчанию все необходимые функции уже выбраны, поэтому просто нажмите «Next».

Подтвердите согласие с лицензионным соглашением, выбрав «I agree».

Опция для создания ярлыков оставлена по умолчанию, поэтому нажмите «Next».

Если хотите создать значок на рабочем столе, установите соответствующую галочку и нажмите «Install».

Дождитесь установки и нажмите «Next».

Завершите установку, нажав «Finish».

Откройте Krita и начните создавать уникальные проекты.

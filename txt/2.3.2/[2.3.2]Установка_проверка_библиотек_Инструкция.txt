Инструкция для установки и проверки библиотек MathPlotLib и Numpy.
Откройте Visual Studio Code.
Откройте встроенный терминал. Для этого нажмите на раздел «Вид» в верхнем меню, затем выберите «Терминал».

В терминале выполните следующие команды для установки библиотек matplotlib и numpy:
pip3 install matplotlib
pip3 install numpy
Дождитесь завершения установки. В терминале будут отображаться сообщения о процессе установки и успешном завершении.
Чтобы убедиться, что библиотеки установлены правильно, вы можете использовать следующие команды:
pip3 show matplotlib
pip3 show numpy
Эти команды выведут информацию о версиях установленных библиотек, если они были установлены успешно.
